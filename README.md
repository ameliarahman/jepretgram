# jepretgram

## Description
Jepretgram is a web application to share picture with caption. Developed using express js, mongoose and vuex

## Step
Server :
- Start database mongo
- npm install
- npm start

Client :
- npm install
- npm run dev

## API
Users
1. http://localhost:3000/api/users/

Post
1. http://localhost:3000/api/posts/
